## 基本配置
```git
//初始化用户名abc
git config --global user.name abc
//初始化电子邮件
git config --global user.email xxxxx@gmail.com
```
## Linux常用命令
```git
//创建文件夹
mkdir file_name

//进入文件夹
cd file_name

//写文件
touch a.txt

//编辑a.txt文件内容,会进入到编写界面
vim a.txt

//按insert按键，开始编写

//编写完成后esc退出编写，:w保存 ，:q回到主窗口 , :wq保存并退出

//查看文件内容
cat a.txt

//清空窗口内容
clear
ctrl+l

//清空文件内容
rm a.txt

//删除文件夹
rm file_name -r
```
